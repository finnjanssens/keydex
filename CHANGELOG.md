# Keydex Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Search keyboard shortcuts across your tools from one Raycast command.
- Auto-discovered sources: ghostty, nvim, tmux, zsh, obsidian, vscode, cursor, zed.
- GUI apps (Arc, Slack, Finder, …) via the macOS Accessibility API, with real app icons.
- Keys rendered as macOS symbols; results searchable by what the shortcut does.
- Cached results for instant reopen; Accessibility source can be toggled in preferences.
